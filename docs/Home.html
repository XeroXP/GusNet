Welcome to GusNet main page.


GusNet is a complete HTTP suite with an special focus on TOR network.


If you are a programmer then GusNet will allow you to use Http services like serving pages or relaying traffic with a minimum code (some times just zero :D ) and connect through the TOR network in seconds, if you are an end user, with GusBundle (a part of GusNet) you can be online in the TOR network with just a few clicks.

If you just want to use GusBundle then go to the Downloads page, else go to the Documentation or Browse the code.

As a programmer, in GusNet you have availables two types of Http servers, an Http proxy, an Http to Socks5 bridge, and a TOR controller.

As an user, using GusBundle you will have acces to the TOR network and your own hidden service up and running in just a few seconds.
